# CodeAlpha_CalculatorProgram

This repository contains **Task 1: Basic - Calculator Program** developed during my **C Programming Internship** at **CodeAlpha**.

## 📌 Project Overview
The goal of this project is to create a robust and continuous command-line arithmetic calculator in C.It allows users to perform basic operations such as Addition, Subtraction, Multiplication, and Division. 

The program utilizes **switch-case statements** for a smooth menu-driven user experience and includes built-in safety checks for standard errors like division by zero.

## 🚀 Features
- **Menu-Driven Interface:** Clear options to choose between different operations.
- **Switch-Case Logic:** Efficient routing of user choices as per internship requirements.
- **Error Handling:** Prevents crashes by checking for division-by-zero errors.
- **Continuous Loop:** Runs indefinitely until the user explicitly chooses to exit the program.

## 🛠️ Tech Stack & Concepts Used
- **Language:** C
- **Core Concepts:** Loops (`while`), Decision-Making (`if-else`, `switch-case`), Data Types (`double` for precise decimal calculations).
